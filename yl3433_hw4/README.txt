Explanations:

Open both effects.py and the effects_tester in your compiler:
1. The module effects.py is already imported
2. Make sure the PPM image to be processed is under the specified directory indicated in the effects_tester.py
3. Please enter one effect at a time. If you want multiple effects on a single image, please run the effects_tester 
   several times.
4. Both programs work perfectly fine on the tetons1new, tetons2new, and tetons3new images. But I am not sure if it     works with the same efficiency on other PPM files. Sorry if it throws you error.
